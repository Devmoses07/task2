STEPS FOR CLONING 
1 first you create a repository in your github account.
2 you copy your repository URL 
3 you clone by using "git clone <url>".
4 Edit your url code by puting your token@ for example "123-rrge@" after your https://
5 Enter the command
6 cd into cloned repo for example <cd task2>
7 create files by using the touch command for example "touch task22"
8 track the files you created by using git add .
9 commit by using git commit -m "message".
10 push by using git push
